Online Quiz Application

Full-stack quiz system with Supabase backend

View on GitHub
Important: This is a Node.js application that requires local setup. It cannot run on GitHub Pages.

Features

  • User authentication with Supabase
  • Timed multiple choice quizzes
  • Admin panel for quiz management
  • Real-time scoring and results
  • Modern responsive UI design

Setup Instructions

  1. Clone: git clone https://github.com/sakthivelmeganathan/quiz-application.git
  2. Install: npm install
  3. Setup Supabase project and update .env file
  4. Run: npm start
  5. Open: http://localhost:3000

Technology Stack

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express

Database: Supabase (PostgreSQL)
